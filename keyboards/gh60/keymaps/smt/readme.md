smt's GH60 Layout
=================

## Quantum MK Firmware

For the full Quantum feature list, see the parent readme.md.

* Standard ANSI layout with Pok3r-inspired function layer
* ESC key led enables when on utility layer
- WASD keys led enables when on function layer
* Neopixel/WS2812 RGB Underglow Support

### Additional Credits

Keymap is based on robotmaxtron's GH60 Layout with RGB underglow mappings, and by various other keymaps available from the QMK Repo for the GH60-SATAN and KC60 keyboards.
